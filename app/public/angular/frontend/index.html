Zu den Tasks

Über die App

Die App basiert auf Jacascript und Node. Es können Tasks erfasst, bearbeitet und gelöscht werden. Erfasste Tasks haben den Status "eröffnet". Durch die Pfeil-Icons könne sie verschoben werden zum Status "in Arbeit" oder "abgeschlossen".

Serverseitig

  • NodeJS - serverseitige Plattform für Webserver
  • CoffeeScript - Javascript Preprocessor
  • Jade - Templateengine
  • Less - CSS Preprocessor
  • Mongoose - Persistenzlayer für Mongo-DB
  • JasmineExpect - Testsuite

Clientseitig

  • AngularJS - MVC-basiertes Framework
  • Bootstrap - CSS-Vorlagen
  • Font-awesome - Icons
  • JQuery - JavaScript Bibliothek
  • Undersc1oreJS - JavaScrupt Bibliothek
